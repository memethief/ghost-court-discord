# Ghost Court helper bot -- Commands

## q [ *role* ... | any ]

**Add myself to the queue**

Status: available

## dq [ *role* ... ]

**Remove myself from the queue**

Status: available

## cq [ *queue name* ... ]

**List queue**

Lists the contents of the given queue or queues, or all queues if none is specified

Status: in progress
